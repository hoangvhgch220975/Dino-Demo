## Design inspirations (awesome-design-md)

Gợi ý cách dùng nhanh:
- Mở `docs/design-md/<brand>/README.md` để tham khảo layout UI, landing pages, components, patterns.
- Khi thiết kế một màn mới, chọn 1–2 brand làm “reference” và note lại trong PR/task.
